LORE DEL SERVER

N_LAND no fue creado por humanos.

Se dice que Nala, una deidad antigua y silenciosa, creó este mundo para divertirse observando jugadores sobrevivir, pelear, construir y destruirse entre ellos. Durante años alteró el mundo a su gusto.

Pero algo salió mal.

Después de una guerra que casi destruye el servidor completo, Nala fue encerrada en otra dimensión.

Se dice que donde se encuentra ahora es el Nether... pero Lacobraa, usando el poder prohibido del queque, bloqueó la dimensión para evitar su regreso.

Muchos creen que fue sellada para siempre... pero otros aseguran que todavía observa el mundo desde fuera.

Tal vez Nala nunca abandonó realmente N_LAND. Tal vez solamente está esperando.
